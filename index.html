Команды Git
Команда Описание
git init создает новый репозиторий Git
git init создает новый репозиторий Git
git clone url клонирует репозиторий по ссылке
git add file добавление в коммит изменений в файле
git add . добавление в коммит изменений во всех файлах
git commit -m "comment" зафиксировать коммит с сообщением
git commit -a -m "comment" добавить изменения в коммит и сразу зафиксировать его
git push url отправляет коммиты в удаленный репозиторий
git status показывает какие файлы были добавлены, а какие изменены относительно предыдущей версии
git log просмотр истории коммитов
git branch branch_name добавляет новую ветку
git checkout branch_name переместиться на ветку branch_name
git checkout -b branch_name создать новую ветку и сразу переместиться на нее
git remote -
